Blankstrap 5 Based on Bootstrap 5

Blankstrap Theme is a free, white labeled, and open source blank theme for RapidWeaver, based on Bootstrap 5. It was never easier to jump start with Bootstrap Development inside RapidWeaver.

It is perfect if you are coming from a traditional RapidWeaver theme and want more freedom. It requires the Stacks Plugin.

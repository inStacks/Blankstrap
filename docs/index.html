Blankstrap Theme

Blankstrap Theme is a free and open source blank theme for RapidWeaver, based on Bootstrap 4.

Download Theme

It was never easier to jump start with Bootstrap Development inside RapidWeaver with this Blankstrap Theme.

Features

Theming

Choose from a range of different themes based on the Bootswatch Project directly from the theme settings inside RapidWeaver.

Navigation

Blankstrap theme contains an optional 1-level RapidWeaver navigation, which can be turned on and off in the theme settings.

Icons

Inbuilt Font Awesome support for nicer icons.

Made to Code

Check the official Bootstrap Documentation for code snippets.
